## [2.2.0] - 2026-03-25

### 新增功能

#### 启动动画
- **SplashScreen 组件** - Logo 动画 + 进度条 + 步骤状态显示
- **静态预加载器** - 纯 HTML + CSS，避免 React 加载白屏
- **应用初始化流程** - 数据库/字体/历史迁移/连接预加载/过期清理
- **字体缓存策略** - localStorage 缓存字体列表，毫秒级启动
- **连接列表预加载** - 启动时加载所有连接到 store，全局共享

#### 命令历史追踪
- **Shell 输出解析** - 从终端输出中自动提取执行的命令
- **多场景支持** - 支持 Tab 补全、上下键历史导航、直接输入
- **提示符检测** - 支持多种 Shell 格式 (`[user@host path]$`、`user@host:path$`、`❯`)
- **控制字符过滤** - 自动过滤 `^C`、`^D`、`^Z` 等控制字符
- **Ghost Text 自动补全** - 根据历史命令显示建议，最多 10 条
- **历史命令弹窗** - 快捷键打开历史命令列表，支持搜索和清空
- **权重算法** - 70% 最近使用 + 30% 使用频率排序

#### 快捷键增强
- **快捷键设置** - 支持清空快捷键（Delete/Backspace）
- **历史命令快捷键** - 自定义快捷键打开历史弹窗

### 改进

- 移除调试日志，优化性能
- 清理未使用代码
- 重新设计连接断开重连逻辑

## [2.1.0] - 2026-03-24

### 新增功能

#### 终端增强
- **xterm.js 升级** - 升级到 xterm.js v6.0.0（@xterm/xterm），支持新特性和性能优化
- **终端滚动条优化** - 更细更美观的滚动条样式
- **终端配置动态更新** - scrollback 等配置实时生效，无需重启

#### 数据库改进
- **Dev/Prod 环境分离** - 开发和生产模式使用独立数据库文件，避免数据污染
- **WAL 模式** - 启用 SQLite WAL 模式，提升并发性能
- **事务保护** - 批量导入操作添加事务保护，防止数据丢失
- **加密密钥改进** - 使用机器唯一标识派生密钥，支持跨会话数据恢复
- **老数据自动迁移** - 自动将老版本加密的密码迁移到新密钥

#### 窗口管理
- **窗口状态插件** - 使用 tauri-plugin-window-state 管理窗口位置和最大化状态
- **多显示器优化** - 修复多显示器 DPI 缩放不一致导致的窗口大小异常

#### UI 改进
- **快速导入粘贴** - 修复快速导入输入框无法使用快捷键粘贴的问题
- **导出连接优化** - 添加全选/取消全选按钮，优化勾选框显示
- **复制信息** - 复制连接信息时显示明文密码
- **最近连接刷新** - 连接成功后最近连接列表立即更新

### 技术改进

- 新增 `tauri-plugin-window-state` 插件
- 新增 `hostname` 依赖用于机器标识
- 改进密码迁移逻辑，支持从 localStorage 和老数据库迁移
- 优化数据库错误处理，记录解析失败日志

## [2.0.0] - 2026-03-22

### 新增功能

#### 终端增强
- **终端分屏** - 支持水平/垂直分屏，可拖拽调整大小，拖拽会话标签创建新分屏
- **多窗口支持** - 拖拽连接标签到窗口边缘创建新窗口，窗口位置自动记忆
- **快捷键系统** - 完整的快捷键支持（分屏、新建会话、关闭会话、搜索等）
- **终端搜索增强** - 支持终端内容全文搜索
- **自动重连** - 断线后自动重连服务器

#### 文件管理
- **文件预览/编辑** - 支持文本文件在线预览和编辑
- **文件搜索** - 远程文件搜索功能
- **文件压缩/解压** - 支持 tar.gz/zip/gz 等格式压缩和解压
- **传输控制** - 支持上传下载暂停/恢复

#### 连接管理
- **SSH 密钥认证** - 支持 OpenSSH 和 PEM 格式密钥文件
- **连接导入/导出** - JSON 格式批量导入导出连接配置
- **连接排序** - 支持拖拽排序和批量管理
- **最近连接** - 记录最近连接历史，快速访问

#### 安全性
- **密码加密存储** - AES-256-GCM 加密存储密码
- **后端数据库** - 连接数据存储在后端 SQLite 数据库

#### MCP 服务
- **MCP 日志面板** - 实时显示 MCP 操作日志
- **日志换行显示** - 支持长命令换行显示
- **命令复制** - 一键复制执行的命令

#### 其他
- **终端主题配置** - 可自定义终端字体、字号、光标样式
- **窗口位置记忆** - 窗口位置在显示前恢复，无闪烁

### 技术改进

- 升级 React 19.x
- 升级 Tauri 2.x
- 使用 russh 0.50 + russh-sftp 2.1
- 前端测试覆盖 135 个测试用例
- E2E 测试支持
